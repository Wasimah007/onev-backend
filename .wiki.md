# Project Summary
The project is a production-ready FastAPI backend for comprehensive timesheet management, user authentication, and role-based access control. It utilizes raw SQL for database interactions, ensuring performance and security while avoiding ORM dependencies. The backend supports multi-tenancy, enabling organizations to manage users, projects, and timesheets effectively.

# Project Module Description
- **Authentication Module**: Manages user registration, login, logout, and token management using JWT.
- **User Management Module**: Handles user profiles, roles, and permissions.
- **Timesheet Module**: Facilitates timesheet entries, including creation, updates, and approval workflows.
- **Project Management Module**: Oversees project definitions, assignments, and billing codes.
- **Leave Management Module**: Manages leave requests and types.

# File Description Inventory
- **main.py**: Entry point for the FastAPI application.
- **config.py**: Configuration settings using Pydantic.
- **db.py**: Database connection and utility functions.
- **auth/**: Contains authentication-related logic and routes.
- **routers/**: API route handlers for various functionalities.
- **services/**: Business logic implementations for different modules.
- **schemas/**: Pydantic models for data validation.
- **utils/**: Helper functions for various tasks.
- **tests/**: Unit tests for the application.

# Technology Stack
- **Backend Framework**: FastAPI
- **Database**: MySQL
- **Asynchronous Database Driver**: aiomysql
- **JWT Library**: python-jose
- **Password Hashing**: passlib (bcrypt)
- **Testing**: pytest
- **Docker**: For containerization and deployment

# Usage
### Installation
1. Clone the repository.
2. Navigate to the backend directory.
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Running the Application
1. **Using Docker** (recommended):
   ```bash
   make docker-up
   ```
2. **Manual Setup**:
   - Run migrations:
     ```bash
     make migrate
     ```
   - Start the development server:
     ```bash
     make dev
     ```
